Configurate Git for your GitHub account

Install Git, with Git BASH and Git GUI
Once install launch Git BASH
Using Git BASH go in the directory where you want to stock your Git folder
=> Command line: cd c:/My_Git_Hub (per example)

Configurate Git to be connected to GitHub (keep the ""):
$ git config --global user.name "your_GitHub_username" 
$ git config --global.email "your_GitHub_mail"



Clone a depository
$ git clone https://github.com/Cyril-Basquin/AWS_Tutorials.git



To add a file for the first time on GitHub

// add the file 
$ git add Test_to_trash.txt

// commit your file for the transfer
// -m is an option to add a comment (do not remove the "")
$ git commit -m "Add a commentary" Test_to_trash.txt

// push the file on your GitHub account
$ git push -u origin master
